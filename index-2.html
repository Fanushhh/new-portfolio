Personal Learning Framework

Choose and customize learning strategies that work for you. Click each framework to explore the steps.

SMART Goals Framework
Structure your learning objectives
+
Specific: What exactly do you want to learn?
Example: Learn Python programming for data analysis
Measurable: How will you track progress?
Example: Complete 3 projects, pass 5 coding challenges
Achievable: Is this realistic given your schedule?
Example: Dedicate 1 hour daily for 3 months
Relevant: Why does this matter to you?
Example: Advance career in data science
Time-bound: When will you achieve this?
Example: Complete by end of quarter
Spaced Repetition System
Optimize retention and recall
+
Initial Learning: Focus on understanding concepts
Example: Read chapter, watch tutorial
Review after 1 day: Quick recall test
Example: Flashcards, practice problems
Review after 3 days: Apply knowledge
Example: Small project, teach someone else
Review after 1 week: Deeper practice
Example: Complex exercises, real scenarios
Review after 2 weeks: Integration
Example: Combine with other skills
Feynman Technique
Learn through teaching and simplification
+
Learn: Study the topic thoroughly
Example: Read materials, take notes
Teach: Explain it in simple terms
Example: Write blog post, record video
Identify gaps: Find what you struggle to explain
Example: Note confusing parts
Simplify: Research and clarify gaps
Example: Use analogies, examples
Repeat: Refine your explanation
Example: Teach again, get feedback
Deliberate Practice
Focus on weak points for rapid improvement
+
Assess current level: Identify strengths/weaknesses
Example: Take diagnostic test, self-evaluate
Target weaknesses: Focus on specific deficits
Example: Practice hardest concepts first
Challenge yourself: Work at edge of ability
Example: Gradually increase difficulty
Get feedback: Immediate, specific corrections
Example: Mentor, peer review, testing
Adjust approach: Modify based on results
Example: Change methods, seek help
Pomodoro Learning
Manage time and maintain focus
+
Plan session: Choose specific learning goal
Example: Complete chapter 3 exercises
Focus 25 minutes: Deep work, no distractions
Example: Phone away, single task
Break 5 minutes: Rest and recharge
Example: Walk, stretch, hydrate
Review progress: What did you accomplish?
Example: Note key insights, questions
Repeat cycle: Long break after 4 rounds
Example: 15-30 minute break

How to Use This Framework

• Start with SMART Goals to define what you want to learn

• Choose 1-2 frameworks that match your learning style and schedule

• Use the checkboxes to track your progress through each step

• Combine techniques - they work great together!

• Adjust as needed - everyone learns differently
